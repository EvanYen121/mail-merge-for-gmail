    Set Up Tracking Server

    1. Open Google Apps Script and create a new project.
    2. Copy and paste the tracking server code from tracker/apps-script.js in this extension folder.
    3. Click Deploy → New deployment → Web App.
    4. Set "Execute as: Me" and "Who has access: Anyone".
    5. Copy the deployment URL and paste it in the Tracking Server URL field.
